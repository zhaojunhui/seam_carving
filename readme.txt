使用说明


编译时因为程序用了openCV，请确保电脑正确安装了openCV。
mac下可以直接运行main或者使用makefile进行编译；
makefile为在mac环境下用cmake生成，cmake工具具有跨
平台性，所以Windows下请在安装了cmake工具的情况下，删掉
makefile、CMakeCache.txt、CMakeFiles,然后打开终端，
进入文件所在路径，运行:
    cmake CMakeLists.txt
    make
即可完成编译。
当然，因为这个程序只有一个源文件，简单按照你的电脑上可以
找到openCV库的方式编译即可，makefile并不必需。
程序会输出足够多的控制台信息，请根据提示输入，由于没有做
输入错误的控制，所以请注意选择参数时不要选择提示中没有的，
是没有默认结果的。
在抠图的时候是有UI界面的，弹出图片后，按下鼠标拖动可以把
经过区域标红，这表示要删除的对象，之后按一下esc，再按下鼠
标拖动可以把经过区域标蓝，这表示要保留的对象，之后按一下esc
程序执行，UI退出。
